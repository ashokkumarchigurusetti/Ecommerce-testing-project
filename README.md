# Ecommerce-testing-project
Comprehensive Manual and Automation Testing of an E-Commerce Web Application using Selenium WebDriver (Python), including test cases, bug reports, and performance observations.

## How to Run the Automation Scripts

1. Clone the repository
2. Install dependencies:
   pip install selenium webdriver-manager
3. Run test files:
   python test_login_updated.py
   python test_add_to_cart_updated.py
   python test_logout_updated.py
